Implement the administrative side of order management, allowing full visibility and control over all orders in the LoomCraft marketplace.

- [x] **Global Order List**:
    - View all orders across all vendors.
    - Show comprehensive status and summary.
- [x] **Full Order Detail View**:

    - Access to all product items and vendor details.
- [x] **Advanced Status Control**:

- [ ] **Admin Notifications**: (To be implemented with Backend integration)
    - Receive alerts for every status change made by vendors.
