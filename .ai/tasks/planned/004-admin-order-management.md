# Task: Admin Order Management & Control

## Status
- Status: Planned
- Priority: High
- Assigned to: Mobile Developer (Kotlin/Compose)

## Description
Implement the administrative side of order management, allowing full visibility and control over all orders in the LoomCraft marketplace.

## Requirements
- [ ] **Global Order List**:
    - View all orders across all vendors.
    - Show comprehensive status and summary.
- [ ] **Full Order Detail View**:
    - Access to all customer information (Name, Address, Phone).
    - Access to all product items and vendor details.
- [ ] **Advanced Status Control**:
    - Ability to override or update any order status (e.g., `Shipping`, `Delivered`, `Disputed`).
- [ ] **Admin Notifications**:
    - Receive alerts for every status change made by vendors.

## Deliverables
- [ ] Admin Dashboard / Order List.
- [ ] Comprehensive Order Detail View.
- [ ] Status Management Interface for Admins.
