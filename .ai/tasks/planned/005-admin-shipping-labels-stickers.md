# Task: Admin Shipping Labels & Sticker Generation

## Status
- Status: Planned
- Priority: Medium
- Assigned to: UI/UX Engineer (AI) & Mobile Developer

## Description
Design and implement a modern, professional shipping label/sticker generation system for Admins. These stickers will be printed and attached to parcels for delivery.

## Requirements
- [ ] **Label Design**:
    - Modern, "rich-looking" layout.
    - Clear typography for delivery addresses.
    - Include: Order ID, Customer Name, Full Address, Contact Number, and Product Summary.
    - Space for branding (LoomCraft logo).
- [ ] **Technical Implementation**:
    - Create a printable PDF or Image format of the label.
    - Option to combine multiple labels into one print sheet if necessary.
- [ ] **User Interface**:
    - A "Print Label" action within the Admin Order Detail view.
    - Preview screen before printing.

## Deliverables
- [ ] UI Component for the Sticker/Label.
- [ ] PDF/Image Generation logic.
- [ ] Print Preview Screen.
